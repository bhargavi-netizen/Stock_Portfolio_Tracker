# Stock Portfolio Tracker (CodeAlpha Task)

## Description
A simple Python program to calculate total investment value based on user input and predefined stock prices.

## Features
- User input for stock name and quantity
- Predefined stock price dictionary
- Total investment calculation
- Saves result to file

## Concepts Used
- Dictionary
- Input/Output
- Loops
- File Handling

## How to Run
python tracker.py
